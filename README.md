# 원신 성유물 기대 레진 계산기

## 프로젝트 개요
원하는 주옵션과 부옵션 수치를 만족하는 성유물을 얻기 위해 필요한 기대 레진을 수학적으로 계산하는 웹앱입니다.

사용자는 성유물 부위, 주옵션, 원하는 부옵션(1~4개) 및 목표 수치를 입력합니다. 앱은 가능한 모든 강화 경우의 수를 전수 조사하여 해당 조건을 만족하기까지 필요한 기대 레진을 정확한 확률 기반으로 계산하여 제공합니다.

## 가정
1. 비경에서 원하는 세트가 나올 확률은 기본 0.5입니다. 비경 1회 보상의 가격은 20 레진입니다.
2. 슬롯 확률은 기본 0.2이며, 5부위를 균등 확률로 가정합니다.
3. 성유물 비경 1회 보상당 평균 5성 성유물 획득량은 1.065개로 둡니다.
4. 모든 5성 성유물은 드롭 시점에 4개의 부옵션이 결정되어 있다고 가정합니다. (3줄 시작은 4번째 옵션이 숨겨진 상태)
5. 부옵션의 초깃값 및 강화 수치는 `maximum_roll` 값에 0.7 / 0.8 / 0.9 / 1.0 중 하나가 균등한 확률로 결정됩니다.
6. 3줄 시작 성유물은 강화 기회가 4회(+8, 12, 16, 20), 4줄 시작 성유물은 5회(+4, 8, 12, 16, 20) 주어지는 것을 반영합니다.
7. 부옵션은 주옵션과 중복될 수 없으며, 등장 확률은 게임 내 실제 가중치(Weight) 시스템을 따릅니다.

## 파일 설명
- `artifact-resin-calculator.html`: 메인 웹앱 파일
*별도의 JSON 파일 없이 모든 데이터는 HTML 내 상수로 포함되어 있습니다.*

이 프로젝트는 다항 분포(Multinomial Distribution)와 가중치 기반 순열(Weighted Permutation)을 사용하여 시뮬레이션 오차 없는 정확한 확률을 계산합니다.

## 실행 방법
HTML 파일을 웹 브라우저에서 열어서 실행할 수 있습니다.
GitHub Pages 또는 정적 웹 호스팅 환경에서 실행할 수 있습니다.

사용자는 다음 순서로 앱을 이용합니다.
1. 꽃, 깃털, 시계, 성배, 왕관 중 원하는 슬롯을 선택합니다.
2. 해당 슬롯에서 가능한 주옵션을 드롭다운에서 선택합니다.
3. 원하는 부옵션을 1~4개 선택하고, 각 부옵션의 목표 수치를 입력합니다.
4. 계산 버튼을 눌러 결과를 확인합니다.

## 출력
앱은 아래 3가지 결과를 단계별로 제공합니다.

1. 원하는 세트, 슬롯, 주옵션을 가진 강화되지 않은 성유물을 얻기 위한 기대 레진값
2. 해당 성유물에서 희망하는 부옵션 종류가 모두 포함될 확률과 기대 레진값
3. 획득한 성유물이 목표 수치까지 강화될 성공 확률과 최종 기대 레진값
